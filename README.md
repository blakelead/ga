# ga

playing with genetic algorithms.

## about

this project aims to evolve a path trough a generated maze by using genetic algorithms.

## build and run

```bash
cmake -D CMAKE_BUILD_TYPE=Debug -S . -B build/Debug
cmake --build build/Debug
./build/Debug/ga
```
